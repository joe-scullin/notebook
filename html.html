Main Root

<!Doctype html> defines what programming language the browser should utilize when reading the code that follows.
<html> the <html> element is the container for all other HTML tags. Everything within these tags will be read as HTML.
lang = "en" the Language attribute specifies the language of the element's content. When placed inside the HTML tag, all of the code within the page will be read as the designated language
(unless, otherwise specified).

Document Metadata

<head> this element contains the page title and metadata
<title> indicates the webpage's name which should appear on the tab
<meta> defines the metadata about your HTML document. Typically this will refer to the document's character set, author, keywords, and other info.
<meta charset = "utf-8">
is the most common use of the meta tag which will encode all of the page data as UTF-8.
charset = "UTF-8" UTF-8: Universal Transformation Format â€“ 8-bit is a variable-length character encoding defined by the Unicode Standard.
UTF-8 is capable of encoding all 2,112,064 valid character code points in Unicode using one to four one-byte (8-bit) code units.
<link> defines the relationship between the current document and an external source
rel = "stylesheet" the rel attribute specifies the relationship between the documents you are linking. In this case, it would be the relationship between the current document and an external CSS style sheet.
Valid on: <link>, <a>, <area>, and <form>

Content / Body

<body> contains all the content which will be viewed on the webpage such as paragraphs, images, links, tables, lists, etc.
<h1> used to indicate headings and their size in HTML. <h1> is the largest and most important, while <h6> is the smallest and least important.
note: Only use one <h1> per page - it should indicate the main heading of the whole page. Also, do not skip heading levels - start with <h1>, then use <h2>, etc.
<p> the paragraph tag indicates that the content within them as paragraphs on a webpage. The browser will automatically add a single blank line before / after each <p> element
<br> inserts a single line break in the text.
It is an empty tag so it does not contain any content or need an end tag.
<a> the anchor tag is used to create links on your page (think of it like anchoring your site to an external site). However, since it is a stand-alone element, it requires other attributes (such as href) to give the information it needs in order for it to function properly.
href = " " hyperlink reference attribute defines the relative or absolute link needed for the hyperlink to direct the user to the desired location
