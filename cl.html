# Shell, Terminal, Command Prompt

Navigation

pwd Print Working Directory: prints the location and pathway to your current directory
ls Lists all items in the current directory or specified directory
    ls   Downloads
    will list the contents of your Downloads folder
cd Change Directory: reverts back to the home directory by default
cd reverts back to home directory of the current user
cd ~ reverts back to user's directory but can be used when writing absolute paths
% cd ~/Downloads
    /Users/joe/Downloads
cd .. goes back one directory level
cd . current directory
cd .. 1 directory back
cd / reverts back to root directory (cores)
cd / specified / path changes directory along specified path
cd - reverts back to previously visited directory

Files / Directories

print / echo prints an argument to the command line
cat print file content to the terminal
touch creates an empty file of specified type
mkdir creates an empty directory
-p makes any necessary parent directories along a specified path
rm removes files
    removing a file using 'rm' will permanently delete it so cannot be recovered
rmdir removes empty directories
    if you want to remove a full directory, you'll need to use the command
    rm -r   Directory\ Name
    which will remove all contents within the target directory
mv moves ( and renames ) files and directories to specified location
    mv can also be used to rename files or directories using the format:
    mv   old_file_name   new_file_name
cp copies files or directories

Data / Storage

zip Create a .zip file from an existing file / directory using the following format
zip     file_name.zip     file_name
-r recursively zips all files in a directory
-u updates the zip archive to add new files
zip -u   file.zip   file2.txt file3.txt
this will add files 2 and 3 to the existing file.zip
-d deletes specified files from .zip
-x excludes specified files while zipping
-m Deletes original files after zipping
*.jpg zip all files within a directory of a specified format as a single file by using the following syntax:
zip   new_zip.zip   *.jpg
now all .jpg files within the directory will be saved as a single .zip file named "new_zip.zip"
individual file output to individually zip all specified files in a directory (as opposed to wrapping them in a single zip file) :
for i in   *.jpg;   do   zip "${i%}.zip"   "$i"; done
where *.jpg indicates the file type you want to zip individually
unzip unzip will list, test, or extract files from a zip archive
tar the difference between zip and tar is that zip is a file compression format while tar is a file archive format. Zip compresses data to reduce its size, while tar combines multiple files into a single archive file, but does not compress the data within it.

Operators

> used to redirect the output of a command to a file. The contents of the file are overwritten each time this operator is used.
    print "Hello" > text.txt
    this will overwrite the contents of file: text.txt and replace it with the string - Hello
>> similar to the > operator, but it appends the output of a command to an existing file, rather than overwriting its contents.
    print "World" >> text.txt
    this will append the string - World - to the already existing content within the file text.txt resulting in
    Hello
    World
