Command-Line

:: Terminal , Shell , etc.

Navigation

pwd Print Working Directory: prints the current location and pathway to your current directory
ls Lists all items in the current directory (Alphabetically by default)
cd Change Directory: reverts back to the home directory by default
cd reverts back to home directory (user)
cd .. goes back one directory level
cd . current directory
cd .. 1 directory back
cd / reverts back to root directory (cores)
cd / specified / path changes directory along specified path
cd ~ begins pathway at user's directory
ex: ~/Downloads = /Users/joe/Downloads
cd - reverts back to previously visited directory

Actions

python3 initiates the python IDE
man shows the user manual for the specified command
- press Q to quit -
whoami prints useranme
print / echo prints and argument to the command line
say computer will say anything you type
sleep make the terminal sleep for a specified time

Flags

-a all hidden items
-l list w/ more detail
-h human readable
-s display the file size
-S sort by size
-t sort by time last modified
-r reverse sort
-F flag the item w/ a marker

Hot Keys

ctrl C Cancels current process
cmd T Opens a new tab
cmd K / clear Kill screen (wipe command terminal)
cmd L clears to previous mark (not undo)
option click move the cursor to a specified spot
ctrl A Goes back to the beginning of the line
ctrl E Goes to the end of the line
ctrl U (undo) Clears to the beginning of the line
ctrl W Cut the word before the cursor
ctrl K Cut the word after the cursor
